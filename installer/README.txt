Shreeji Auto Service Admin - Installation

1. Double-click setup.bat to install.
2. Windows may ask for administrator permission. Click Yes.
3. After install, open the app from the desktop shortcut or the Start Menu.
4. The app requires an internet connection to work because it connects to the Neon cloud database.
5. To uninstall, double-click uninstall.bat.

Do not delete files from the installed folder manually. Use uninstall.bat when removing the app.
